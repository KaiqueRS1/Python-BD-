# Estudos de Python para Engenharia de Dados

Este repositório registra minha evolução nos estudos de Python, com exercícios, correções e pequenos projetos voltados para Engenharia de Dados.

## Objetivo

Desenvolver uma base sólida em Python e aprender a utilizar a linguagem para manipulação, tratamento e processamento de dados.

## Conteúdos estudados

- Variáveis e tipos de dados
- Entrada e saída de informações
- Estruturas condicionais
- Laços de repetição
- Validação de dados
- Funções
- Manipulação de arquivos
- Pandas
- ETL
